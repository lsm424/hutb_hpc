## 1. 指标框架概述
- 面向效率提升与稳定体验的指标体系，覆盖使用、性能与运营三维度。

## 2. 北极星指标定义
- 问题定位时间缩短比例（管理员/科研用户）：目标30%。

## 3. HEART / AARRR 指标体系

### HEART
- Happiness：满意度调查与NPS。
- Engagement：关键视图使用率（分区/作业/节点），日均进入次数。
- Adoption：新用户7日留存，功能首用率（筛选/导出/订阅）。
- Retention：周/月活留存，回访率。
- Task Success：筛选成功率、导出成功率、告警订阅成功率。

### AARRR
- Acquisition：到达平台的用户数（登录成功）。
- Activation：完成首屏浏览与一次筛选的用户比例。
- Retention：周/月留存与回访频次。
- Revenue：内部价值度量（问题定位时间下降、工单量减少）。
- Referral：内部推荐度（NPS与他人引荐）。

## 4. 功能级评估指标

| 模块 | 指标 |
|---|---|
| 分区看板 | 数据延迟P95（15s）、刷新成功率、排序使用率 |
| 作业视图 | 筛选成功率、联动点击率、详情展开率 |
| 节点视图 | 健康状态命中率、异常节点识别率 |
| 用户统计 | 在线统计准确率、时间窗口切换成功率 |
| 搜索筛选 | 搜索命中率、组合筛选使用率 |
| 导出 | 导出成功率、文件下载完成率 |
| 告警订阅 | 订阅成功率、告警及时送达率 |

## 5. 指标监测计划
- 埋点：分区卡片点击、筛选变更、详情展开、导出、订阅、登录。
- 后端：接口耗时、错误率、数据延迟；队列/缓存命中率。
- 报告：周报/月报；异常波动自动告警（IM/邮件）。
